# Shama-Brothers-SoftGrid-Internship

## Project Overview
A modern, premium website for Shama Brothers BBQ, showcasing the menu, story, and contact information with vibrant design, smooth animations, and a dark‑mode theme.

## Responsive Compatibility
The site is fully responsive across:
- **Mobile Devices** (320 px – 599 px)
- **Tablets** (600 px – 899 px)
- **Laptops** (900 px – 1199 px)
- **Desktop Screens** (1200 px +)
See `RESPONSIVE_DESIGN_GUIDE.md` for detailed breakpoints and testing checklist.
